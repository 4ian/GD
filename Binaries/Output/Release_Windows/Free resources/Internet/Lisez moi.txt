Envoi de données par internet pour Game Develop
-----------------------------------------------

Avant de pouvoir envoyer des données sur internet, il vous faut mettre en place votre site web. Vous aurez besoin d'un site supportant PHP ( renseignez vous auprès de votre hébergeur en cas de doute ).

Ces fichiers sont à placer sur votre site.

view.php sert à voir le tableau récapitulatif ( tableau des scores, tableau des meilleurs temps... )
send.php permet d'envoyer les données au site. Vous devez renseigner cette page dans l'éditeur.
config.php permet de personnaliser les messages, l'ordre d'affichage du tableau, et les entête du tableau.
datas.txt stocke les données. Si le fichier n'existe pas, il sera créé.

Pour plus d'informations, consultez l'aide de Game Develop.
