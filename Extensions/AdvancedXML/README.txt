Advanced XML is a free implementation of TinyXML for GDevelop.

Source code organisation
========================
	- res 			-> Contains images needed by the extension (this folder's files should be copied in GDDIR/res/AdvancedXML folder)
	- src			-> Contains source code and Code::Block project file
	- testSamples 		-> Contains some examples for testing purpose
